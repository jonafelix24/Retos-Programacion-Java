# ENCUENTRA EL PRIMER NÚMERO NO CONSECUTIVO

**Nivel:** 1
<br>
**Tags:** Básico, Arrays
<br>
**Fuente:** [Codewars](https://www.codewars.com/kata/58f8a3a27a5c28d92e000144)

## DESCRIPCIÓN:

El reto es encontrar el primer elemento de un array que no sea consecutivo. 

Un número no consecutivo en este reto es aquel elemento que no es exactamente 1 más que el elemento anterior del array.

Si todos los elementos del array son consecutivos entonces devuelve un `null`.

**Nota:**
- Todos los elementos del array son números.
- El array siempre tiene al menos 2 elementos.
- Todos los números del array son únicos.
- Los elementos del array siempre van en orden ascendente.

## Ejemplos

**Input:** `[1,2,3,4,6,7,8]`
<br>
**Output:** `6`
